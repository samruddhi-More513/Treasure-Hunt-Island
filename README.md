# 🏝️ Treasure Island Game (Python)

## 📌 Project Overview

A command-line based interactive game built using Python.
This is my second project in my Python learning journey, consisting of **100+ lines of code** and focused on improving logic, structure, and user interaction.

---

## 🎮 Features

* Decision-based gameplay (multiple outcomes)
* Input validation
* Replay functionality
* Loading/animation effects using `time.sleep()`
* Same-line output using `print(end="")`
* Real-time output handling using `flush=True`
* Screen clearing using `os.system('cls')`
* ASCII art for enhanced CLI experience

---

## 🧠 Concepts Used

* Loops (`while`)
* Conditional statements (`if-elif-else`)
* Functions
* User input handling
* Output formatting and control
* Basic CLI UX improvements

---
